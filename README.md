# AI.md

## Interação 1

### O que foi solicitado para a IA

Solicitamos auxílio para criar a estrutura inicial da landing page do projeto SIGMA utilizando HTML e CSS.

### O que a IA retornou

Foi sugerida uma estrutura contendo barra de navegação, seção principal (hero), seções informativas e rodapé, além de estilos básicos para organização visual da página.

### O que foi alterado ou rejeitado e o motivo

A estrutura foi adaptada para atender ao tema do projeto SIGMA. Foram alterados textos, imagens, cores e organização de algumas seções para representar melhor o monitoramento de enchentes e terremotos.

---

## Interação 2

### O que foi solicitado para a IA

Solicitamos ajuda para implementar a validação do formulário de contato utilizando JavaScript.

### O que a IA retornou

Foi fornecido um código capaz de validar os campos de nome, e-mail, assunto e mensagem, impedindo o envio quando existissem campos inválidos.

### O que foi alterado ou rejeitado e o motivo

Foram ajustadas algumas mensagens de erro e a aparência visual para manter a identidade do site.
